# swift-rfc-2822

Swift implementation of [RFC 2822: Internet Message Format](https://www.rfc-editor.org/rfc/rfc2822.html)

## Installation

### Swift Package Manager

Add this to your Package.swift:

```swift
dependencies: [
    .package(url: "https://github.com/swift-web-standards/swift-rfc-2822.git", branch: "main")
]
```
